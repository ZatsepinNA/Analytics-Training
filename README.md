# Роадмап: Data Analyst Стажировка (Январь – Май 2026)

**Цель**: К концу мая 2026 быть конкурентоспособным кандидатом на летние стажировки в крупные российские компании (Сбер, Тинькофф, Яндекс, VK, Ozon и др.) в роли Junior Data Analyst / BI Analyst.

**Формат обучения**:  
- 15–20 ч/неделю  
- Акцент на практику + портфолио  
- Без готовых ответов — только наводящие вопросы и фидбэк  
- Двухнедельные спринты с адаптацией

---

##  Фазы

###  Фаза 1: Январь – начало февраля (Недели 1–4)  
**Фокус**: Глубокий SQL + Основы аналитического Python

**SQL**:
- Завершить SQL Academy или аналог (PostgreSQL Tutorial)
- Освоить:  
  - Window Functions (`ROW_NUMBER`, `RANK`, `LAG/LEAD`)  
  - CTE и рекурсивные запросы  
  - Агрегация по времени (rolling, cohorts)  
  - Базовая оптимизация (`EXPLAIN`, индексы — на уровне понимания)
- Практика: LeetCode (Medium), HackerRank, StrataScratch

**Python**:
- Перейти от базового Python к аналитическому стеку:  
  - `pandas`: фильтрация, группировка, merge, pivot, обработка NaN  
  - `matplotlib` / `seaborn`: bar, line, scatter, histogram
- Ресурсы: Kaggle Learn (Pandas, Data Visualization)

 **Итог**: решаешь сложные SQL-задачи + загружаешь CSV → анализируешь → визуализируешь на Python.

---

###  Фаза 2: Февраль – март (Недели 5–9)  
**Фокус**: Power BI + продвинутая аналитика + первый кейс

**Power BI**:
- Подключение источников (CSV, Excel, SQL)
- DAX: меры, CALCULATE, FILTER, TIME INTELLIGENCE
- Построение дашбордов с KPI, фильтрами, иерархиями
- Ресурсы: Microsoft Learn, Guy in a Cube (YouTube)

**Аналитика**:
- Метрики: DAU/MAU, Retention, LTV, Conversion Funnel
- Cohort-анализ (на SQL + Python)
- A/B-тесты: гипотезы, p-value, t-test (на уровне интерпретации)

**Первый кейс**:
- Датасет: Online Retail / Superstore / Mobile App Events
- Этапы: EDA → метрики → выводы → дашборд в Power BI → презентация (3–5 слайдов)

 **Итог**: первый проект в портфолио (GitHub + Power BI).

---

###  Фаза 3: Апрель (Недели 10–13)  
**Фокус**: Углубление + второй кейс + подготовка к собеседованиям

**Навыки**:
- SQL: работа с большими таблицами, оптимизация
- Python: автоматизация отчётов, работа с API (`requests`, JSON)
- Power BI: параметры, bookmarks, storytelling

**Второй кейс**:
- Тема: "Как увеличить retention?" / "Оценка эффективности акции"
- Включает: гипотезы, A/B-тест, простой прогноз (скользящее среднее)
- Публикация: GitHub + Power BI Service / Tableau Public

**Подготовка к собеседованиям**:
- Типовые вопросы:  
  - «Как посчитать retention?»  
  - «Что делать, если метрика упала?»  
  - «Как проверить эффект изменения UI?»
- Framework ответов: проблема → данные → гипотезы → решение

 **Итог**: 2 проекта, уверенность в кейсах, готовность к техническим заданиям.

---

###  Фаза 4: Май (Недели 14–18)  
**Фокус**: Финальная шлифовка + подача заявок

- Доработка резюме (акцент на проекты и навыки)
- Создание LinkedIn-профиля
- Подача в 10–15 компаний
- Мок-собеседования (по запросу)
- Оттачивание питч-презентации проектов

 **Итог**: активная подача на стажировки с сильным портфолио и чёткой позицией.
