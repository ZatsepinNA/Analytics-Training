# Подготовка к стажировке Data Analyst

План обучения и трекинг прогресса для входа в профессию к маю 2026 года.

## Цель
Стажировка в крупной компании (Сбер, Тинькофф, Яндекс, Ozon) на позицию Junior Data Analyst / BI Analyst.

## Технологический стек
- SQL: PostgreSQL, оконные функции, CTE, оптимизация
- Python: pandas, matplotlib, seaborn
- BI: Power BI, DAX, моделирование данных
- Инструменты: Git, DBeaver, Jupyter
